ที่มา

ประมาณปี 2022 ได้เรียนรู้การเล่นดนตรีจากการเขียนโค้ด แล้วไปพบกับ RepRapRingtone และ rtttl-play ซึ่งเป็นโน้ตในรูปแบบที่เรียนกว่า RTTTL (สามารถอ่านเพิ่มได้ที่ Ring Tone Text Transfer Language จึงได้นำมาเรียนรู้และประยุกต์สร้างเป็นเว็บสำหรับเล่นเสียงจากโน้ตดนตรีไทย โดยเวอร์ชันแรกจะเป็นตัวแปลงโน้ตดนตรีไทยไปเป็นโน้ตระบบ RTTTL โดยเป็นโปรเจคแรกที่ทำให้ได้เรียนรู้เรื่องของ javascript แบบที่มีความสับซ้อนขึ้นมา สามารถเยี่ยมชมได้ที่ warut92.github.io/rtttl-th

การทำงาน

เป็นการพิมพ์ตัวโน้ตดนตรีไทยในระบบ ดรมฟ หรือ ระบบ 1-9 จากนั้นโปรแกรมจะแปลงโน้ตให้เป็นระบบ RTTTL และจะสังเคราะห์เสียงพื้นฐานจาก Web Audio API ออกมา (สำหรับการเล่นในโทรศัพท์มือถือ ต้องปิดโหมดการรบกวน)
